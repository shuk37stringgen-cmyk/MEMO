# カラーメモ (Raku-Memo) - 思考をキャンバスに並べる直感型メモダッシュボード

# カラーメモ

## アプリは右のアバウトから確認できます。



## 概要

「カラーメモ」は、会議中のホワイトボードのように、メモを画面上の好きな位置へドラッグ＆ドロップで自由に配置・整理できるメモアプリケーションです。

既存のメモアプリでは、テキストが一覧で並ぶだけのものが多く、「どこに何があるか」を視覚的に把握しづらいと感じていました。

そこで本アプリでは、メモの**位置・大きさ・色**を自由に変更できるようにし、情報の重要度やカテゴリを視覚的に整理できるダッシュボード形式を採用しました。フォルダによる管理と自由配置を組み合わせることで、直感的な情報整理を実現しています。

---

# こだわったポイント

## 手触り感を意識したドラッグ＆ドロップ

メモは画面上の好きな場所へ配置できます。

さらに、見えないグリッドへ自動で吸着する「スナップ機能」を実装し、自由度を保ちながらも整列しやすい操作感を実現しました。

---

## ゲームのセレクト画面を意識したUI設計

ゲーム制作で培ったUI設計の考え方を活かし、メモのサイズを自由に変更できるようにしました。

重要な情報は大きく、補足情報は小さく配置することで、一覧を開いた瞬間に情報の優先順位が伝わるレイアウトを目指しています。

---

## 思考を止めない視認性

素早くメモを取る体験を重視し、あえて文字色の変更機能は設けませんでした。

その代わり、背景色が白・黒・鮮やかな色であっても文字が読みやすくなるよう、自動でテキストにアウトラインを付与する仕組みを実装しています。

# 開発を通して学んだこと

本プロジェクトは、HTML・CSS・JavaScriptを用いた本格的なフロントエンド開発への挑戦でした。

AIを壁打ち相手として活用しながら、UI設計や機能のロジックを自分で考え、試行錯誤を繰り返して実装しました。

特に、ドラッグ＆ドロップ、グリッドスナップ、当たり判定などの機能を実装する中で、「思い描いたデザインをコードで形にする面白さ」を実感しました。

今回の開発を通して、プログラミングは単に機能を実装するだけでなく、ユーザー体験を設計する手段でもあると学びました。今後もUI/UXと実装の両面を磨き、頭の中のアイデアを高い解像度で形にできるエンジニア・クリエイターを目指していきます。

# 今後追加したい機能

現在は個人利用を前提としていますが、今後はチームでの利用も視野に入れています。

* フォルダごとの Public / Private 管理
* メモ単位でのパスワードロック機能
* 社内向け共有フォルダ機能
* 複数人で利用できる共有ダッシュボード

個人のアイデア整理からチームでの情報共有まで対応できる、より柔軟なメモアプリへ発展させたいと考えています。
